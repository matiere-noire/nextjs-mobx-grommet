<h1>NextJS + Mobx + Axios + Grommet</h1>

The perfect ReactJS stack according to us. 

## Features

- ⚛️ React as the view.
- MobX as state.
- Next.js /w Express server (SSR).
- Axios as the http client.
- SASS via `styled-jsx`.
- GZIP by `compression`.
- `hpp` & `helmet` for security. Also forces `https`.
